# HealthPress Lite

A lightweight healthcare and business WordPress theme.

## Features

- Responsive design
- Tailwind CSS
- Custom Logo
- Custom Menus
- Blog Support
- WordPress Theme Customizer
- Gutenberg Ready

## Installation

1. Download the theme.
2. Upload to `wp-content/themes/`.
3. Activate from WordPress admin.

## Development

```bash
npm install
npx @tailwindcss/cli -i ./assets/css/input.css -o ./assets/css/output.css --watch
